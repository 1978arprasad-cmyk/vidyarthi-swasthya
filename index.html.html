☁️ Real-Time Sync Status — Supabase

Real-time sync built in
Every change replicates across devices in < 1 second.
✅ Pre-Configured — No Setup Needed

This app ships with Supabase already wired in. The same Supabase project is used on every device — phone, tablet, desktop. There is nothing to paste, nothing to install.

Project: tswkudgfmsktqfbfychc.supabase.co
How real-time sync works
1
Login on Device A
Add a student or health record. Saved to Supabase Postgres immediately.
2
Postgres triggers a Realtime event
Supabase broadcasts INSERT / UPDATE / DELETE to every device subscribed to your school's channel.
3
Device B updates instantly
No refresh, no re-login. New data appears in the list under one second.
4
Login from any device with the same code+PIN
School data lives in the cloud, not on the device. Open on a new phone → same students, same records.
🔒 Privacy: Each school's data is keyed by school code. Only someone with the code+PIN can see it. 💡 Hosting: Free Supabase tier — 500 MB database, 5 GB bandwidth/month.
